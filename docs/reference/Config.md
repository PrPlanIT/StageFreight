| `protect` | `retention.protect` | []string | Yes | ‚Äî | Tag patterns that are never deleted. |
| `tags` | `tags` | []string | No | ‚Äî | Tag templates resolved against version info. `kind: registry` only. |
| `file` | `file` | string | No | ‚Äî | Path to the README file. `kind: docker-readme` only. |
| `link_base` | `link_base` | string | No | ‚Äî | Base URL for relative link rewriting. `kind: docker-readme` only. |
| `spec_files` | `spec_files` | []string | No | ‚Äî | Component spec file paths. `kind: gitlab-component` only. |
| `catalog` | `catalog` | bool | No | ‚Äî | Enable GitLab Catalog registration. `kind: gitlab-component` only. |

**`kind` allowed values:** `registry`, `docker-readme`, `gitlab-component`, `release`

**`when.events` allowed values:** `push`, `tag`, `release`, `schedule`, `manual`, `pull_request`, `merge_request`

> The `when` block controls routing: all non-empty fields must match (AND logic).

**Example:**

```yaml
targets:
  - id: dockerhub-stable
    kind: registry
    build: myapp
    url: docker.io
    path: myorg/myapp
    tags: ["{version}", "latest"]
    when: { git_tags: [stable], events: [tag] }
    credentials: DOCKER
```
